CSS Selectors

Defines the elements to which a set of CSS rules apply

CSS selectors are categorized into:

  • Simple selectors
    • Type / Tag selector
    • Class selector
    • Id selector
    • Universal selector
    • Attribute selector
  • Combinators
    • Adjacent sibling combinators
    • General sibling combinators
    • Child combinators
    • Descendant combinators
  • Pseudo classes
    • :active
    • :checked
    • :disabled
    • :first-child
    • nth-child
    • :last-child
    • :hover
  • Pseudo elements
    • ::after
    • ::before
    • ::cue
    • ::first-line
    • ::first-letter

Type selector

Selects all elements that match the given node name. It selects all elements of the given type within a document

syntax:

element { style properties }

Class selector

Selects all elements that have the given class attribute

syntax:

element [ attr : attr value ] { style properties }
